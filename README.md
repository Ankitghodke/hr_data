# hr_data
Analyzed HR employee data using Python to identify attrition trends across departments and job satisfaction levels. Performed data cleaning, exploratory data analysis, and visualizations to uncover key workforce patterns and support data-driven HR decisions.
This project focuses on analyzing HR employee data to understand attrition patterns and workforce behavior. The goal is to identify key factors influencing employee attrition using Python-based data analysis techniques.
